# FYP Ch13: Multichannel Learning Trajectory Studio

This Python-first FYP translates Chapter 13 into a deployable browser package. It keeps the real engagement and achievement workbook locally, clusters paired learner paths, and supports manual multichannel input without requiring R.

## Run Order

1. Run `./run/run_backend.ps1`
2. Run `./run/run_frontend.ps1`
3. Open `http://127.0.0.1:8513/frontend/index.html`
