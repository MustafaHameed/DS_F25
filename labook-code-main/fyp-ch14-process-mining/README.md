# FYP Ch14: Learning Process Mining Studio

This Python-first FYP translates Chapter 14 into a browser-ready process-mining package. It sessionizes the real Moodle event log, trims the dominant activity flow, compares high and low achiever process patterns, and supports manual action-path scoring.

## Run Order

1. Run `./run/run_backend.ps1`
2. Run `./run/run_frontend.ps1`
3. Open `http://127.0.0.1:8514/frontend/index.html`
