# Project Brief

## Domain

Educational data mining and model-based student profiling.

## System Summary

This project translates the Chapter 09 latent-profile concept into a Python-based FYP. It discovers hidden engagement profiles from the chapter dataset and provides a browser interface where a user can manually enter engagement scores and inspect soft profile probabilities.

## Core Modules

1. CSV ingestion from the engagement dataset.
2. Feature selection and QMD-aligned renaming.
3. Gaussian Mixture modeling with diagonal covariance.
4. Posterior probability and entropy analysis.
5. Manual-input probability estimation in the browser.

## Deliverables

- reproducible Python backend
- manual-input web frontend
- profile assignments and probability artifacts
- FYP-facing documentation
