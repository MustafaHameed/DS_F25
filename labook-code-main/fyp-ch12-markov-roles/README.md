# FYP Ch12: Role Transition Markov Studio

## Project Title

Role Transition Markov Studio for Longitudinal Learner-Role Sequences

## Summary

This Python-first FYP translates Chapter 12 into a browser-ready package. It models observed role transitions, estimates sequence clusters inspired by mixture Markov models, and lets a user test a full role sequence manually without requiring R.

## Run Order

1. Run `./run/run_backend.ps1`
2. Run `./run/run_frontend.ps1`
3. Open `http://127.0.0.1:8512/frontend/index.html`
