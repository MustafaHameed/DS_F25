# Project Brief

## Domain

Longitudinal learning analytics and educational trajectory analysis.

## System Summary

This project translates the Chapter 11 VaSSTra method into a Python-first FYP. It converts weekly behavioral indicators into latent engagement states and then groups full state sequences into learner trajectories that can be explored and tested manually in the browser.

## Core Modules

1. Longitudinal data loading from a local raw copy.
2. Course-wise standardization of behavioral indicators.
3. Gaussian-mixture state discovery.
4. Trajectory clustering across eight time points.
5. Manual-input web demo for trajectory matching.
