# Project Brief

## Domain

Educational data mining and network analytics.

## System Summary

This project translates the Chapter 08 clustering concepts from the original QMD into a Python-based FYP. It groups students according to their network centrality patterns and provides a browser interface where a user can test manual centrality inputs and see the nearest student-cluster profile.

## Core Modules

1. CSV ingestion from the local MOOC centrality dataset.
2. Standardization and hierarchical clustering in Python.
3. Cluster validation and profile extraction.
4. Manual-input cluster assignment using centroid-distance comparison.
5. Web-based visualization of cluster structure and members.

## Deliverables

- reproducible Python backend
- manual-input web frontend
- cluster assignments and profile artifacts
- FYP-facing documentation
