BSIT Final Year Project Collection

Machine Learning and Data Science FYP Portfolio

This portfolio repackages the book/tutorial chapters into project-style deliverables. Each FYP keeps the original QMD chapter as the conceptual source, runs from Python backends, uses local copied datasets, and exposes the analysis through a project-specific frontend with manual input.

Chapters targeted: 07 to 21
Backend runtime: Python scripts + notebooks
Frontend rule: Manual input required
Implemented FYPs: Chapters 07 to 14
Verified notebook coverage: Chapters 07 to 21
